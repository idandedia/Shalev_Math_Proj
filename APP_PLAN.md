# תוכנית עבודה – אפליקציית תרגילי חשבון לכיתה ב'

## מטרת האפליקציה
אפליקציית תרגול אינטראקטיבית ומהנה לילדים בכיתה ב', בעברית, עם תרגילי חיבור וחיסור עד 100.

## דרישות ליבה
1. **שפה עברית מלאה** בכל הממשק.
2. **20 שלבים** – בכל סיום שלב מתקדמים לשלב הבא.
3. **תרגילים רנדומליים יחסית** לחיבור/חיסור עד 100.
4. **ממשק אינטראקטיבי ונעים לילדים**.
5. **משוב שגיאה יפה וברור**.
6. **האפליקציה חייבת לעבוד בפועל**.

## בחירת טכנולוגיה
- **Python + Streamlit**
  - מאפשר בנייה מהירה של UI אינטראקטיבי.
  - פשוט להפעלה מקומית.
  - מתאים להדגמה ולשימוש מיידי.

## מבנה שלבים מוצע
- שלבים 1–5: מספרים קטנים יחסית.
- שלבים 6–10: טווח בינוני, יותר ערבוב חיבור/חיסור.
- שלבים 11–15: טווח גבוה יותר עד 100.
- שלבים 16–20: קושי מאוזן, כולל מספר תרגילים בכל שלב.

## UX לילדים
- כותרות גדולות וברורות.
- כפתורים בולטים.
- הודעות הצלחה/עידוד.
- הודעות שגיאה ידידותיות ומעוצבות.
- דמויות/אייקונים (אמוג'י) להמחשה ומשחקיות.

## לוגיקת משחק
- בכל שלב יש כמה תרגילים (לדוגמה: 3–5 לפי קושי).
- רק לאחר פתרון נכון של כל תרגילי השלב – מעבר לשלב הבא.
- שמירת מצב נוכחי ב-Session State.
- מסך סיום חגיגי בשלב 20.

## בדיקות והרצה
1. התקנת דרישות (`pip install -r requirements.txt`).
2. הרצת האפליקציה (`streamlit run app.py`).
3. בדיקה ידנית:
   - RTL תקין בעברית.
   - מעבר שלבים תקין.
   - תרגילים עד 100 בלבד.
   - הודעות שגיאה/הצלחה מוצגות יפה.
